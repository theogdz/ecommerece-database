Steps to demonstrate the code:

Step 1:
Create a database called Online Grocery Store on pgAdmin.

Step 2:
Using the query tool, copy and paste the "Relational Schema" file. If this doesn't work, use the "Database dump" file instead.

Step 3:
Refresh the database.

Step 4:
Using the query tool, copy and paste the "Sample Data" file to populate the database.

Step 5:
Open Eclipse and click File->Import->General->Existing Projects Into Workspace.

Step 6:
Browse to the location of the project folder and click finish.

Step 7:
Download the code and change the password to access the database from the java program.

Step 8:
Test the code!
